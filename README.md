## Overview

**VizKit** is a lightweight Windows toolkit written entirely in **Batch**.

It provides a single interactive launcher that exposes dozens of built-in Windows utilities, management consoles, diagnostic commands, and productivity shortcutsвЂ”all from one menu.

No installation.

No dependencies.

No external software required.

## Running

Clone the repository:

```bash
git clone https://github.com/Nightmare-De9/VizKit.git
```

Or download the latest release.

Run:

```text
VizKit.bat
```

For the best experience, run **as Administrator** when you need access to privileged Windows management tools.

## Philosophy

VizKit follows a few simple principles:

* Keep everything portable.
* Prefer native Windows capabilities.
* Avoid unnecessary dependencies.
* Stay lightweight.
* Be easy to modify and extend.
